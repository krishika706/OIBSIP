# OIBSIP
This repository contains the task 1 and task 2 completed during the Oasis Infobyte Web Development Internship (OIBSIP). The projects are developed using HTML, CSS, and JavaScript, focusing on building responsive layouts, improving design skills, and strengthening front-end development concepts.
## This project focuses on:
- Creating structured sections
- Adding headers and footers
- Proper alignment and spacing
- Using color palettes and styling
- Ensuring elements do not overlap
